# **App Name**: Amigo Secreto Organizer

## Core Features:

- Participant List Management: Allows users to create and manage a list of participants for the Amigo Secreto, including names and descriptive information. Can be edited until the game starts.
- Value Limit Setting: Enables setting a maximum value for the gift to be offered.
- Secret Assignment: Automatically assign participants to each other to start the Amigo Secreto.
- Individual Links: Each participant gets a link showing who they're assigned to.

## Style Guidelines:

- Primary color: Vivid purple (#9F50E3) for a cheerful and engaging atmosphere.
- Background color: Light pastel lavender (#E3D0F7).
- Accent color: Bright pink (#E35097) for emphasis on key interactive elements.
- Body and headline font: 'PT Sans', a modern humanist sans-serif for readability and friendly appeal.
- Use festive, gift-themed icons throughout the app to enhance visual interest.
- Clean and simple layout with clear sections for participant input, value settings, and assignment results.
- Subtle animations on successful submissions.
